Lowongan Pekerjaan IT

Software Developer

Mengembangkan perangkat lunak dengan kebutuhan klien atau perusahaan.

  • Keahlian: Python, Java, C++, SQL
  • Pengetahuan: Agile/Scrum, Debugging
Data Scientist

Menganalisis data untuk wawasan bisnis dan pengambilan keputusan.

  • Keahlian: Statistik, Python, R
  • Pengetahuan: Big Data, Visualisasi Data
DevOps Engineer

Mengotomatisasi alur kerja pengembangan dan operasi.

  • Keahlian: CI/CD (Jenkins, GitLab CI), Docker, Kubernetes
  • Pengetahuan: Cloud (AWS, GCP, Azure)
Cybersecurity Specialist

Melindungi jaringan dan sistem dari ancaman keamanan siber.

  • Keahlian: Pengujian penetrasi, firewall
  • Pengetahuan: ISO, NIST, scripting keamanan
UI/UX Designer

Merancang pengalaman pengguna yang intuitif dan menarik.

  • Keahlian: Figma, Sketch, Adobe XD
  • Pengetahuan: Prinsip desain, user testing
Network Administrator

Mengelola dan memelihara infrastruktur jaringan perusahaan.

  • Keahlian: Cisco, VPN, DHCP
  • Pengetahuan: Troubleshooting, monitoring jaringan
